博客功能和依赖

Posted on 2020-03-07 Edited on 2020-04-23 In Technical , 工具教程

本搭建过程基于MacOS，具体搭建方法网上资源已经很多了，但是有些已经过时。建议还是直接查看官方文档，因此这里只列出功能使用时候用到了哪些工具，基本照着别人的quick start来一下就成。

功能和工具

  • 静态资源服务器和域名: github page
  • 静态博客资源: hexo
  • 博客主题: NexT
  • 博客评价: gitalk
  • 博客访问记录: GoogleAnalytics, 百度统计
  • 图床: github 修改了服务协议，现在直接把图片放在github上了。之前使用七牛云对象存储
  • SEO优化: Next SEO, Hexo NexT 主题SEO优化指南

已知问题未解决

  • github page把百度封了，因此百度爬虫无法发现
